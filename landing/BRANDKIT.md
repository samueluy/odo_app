# Odo Brand Kit

## Brand Positioning

Odo is a playful performance layer for knowledge workers on macOS.
It turns everyday cursor movement into measurable progress and momentum.

## Tone of Voice

- Energetic, concise, confident.
- Friendly with a premium software finish.
- Focused on momentum, consistency, and craft.

## Core Message

- Primary line: Turn cursor miles into momentum.
- Product line: A menu bar odometer for macOS.
- Conversion line: One-time purchase, no subscription.

## Color System

- Ink: #121217
- Surface: #F3F3F4
- Haze: #E8E8EB
- Accent: #FF7A1A
- Accent Soft: #FFD7B8
- Line: #D8D8DC

## Typography

- Display: Sora (800, 700)
- Body: DM Sans (500, 400)
- Numerics: use tabular/monospace style where possible for readouts

## Visual Direction

- Bright neutral backgrounds with subtle texture or grid treatment.
- Rounded geometry and high-contrast black buttons.
- Orange accents only for action, momentum, and achievement moments.
- Use large, compressed headlines and generous spacing.

## UI Tokens

Defined in [app/globals.css](app/globals.css):

- --ink
- --surface
- --haze
- --accent
- --accent-soft
- --line
